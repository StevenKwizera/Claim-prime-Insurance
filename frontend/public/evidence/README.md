# Prime Insurance — demo evidence media

These files power the **Evidence Gallery** previews (auto, health, property, billing).

| File | Claim theme |
|------|-------------|
| `auto-*.jpg` | Motor / collision damage |
| `health-*.jpg` | Medical & dental claims |
| `property-*.jpg` | Flood, fire, storm, burglary |
| `billing-*.jpg` | Receipts & pharmacy |
| `identity-*.jpg` | ID verification |
| `video-*.mp4` | Dashcam, walkthroughs, drone, consultation |

Re-download assets: `npm run evidence:fetch` from the project root.
